Matrices and its operations

矩陣

assume F is a field and A is a \(m\times n\) matrix over F
each element \(a_{ij} \in F\) and \(a_{ij}\) is scalar

  • Real matrix: \(F=R\)
  • Complex matrix: \(F=C\)
  • Zero matrix: 全部元素皆為0，記為\(O\)
  • Negative matrix: -A = [-\(a_{ij}\)]
  • Identity matrix: 對角線元素全都是1，其他元素皆0，記為\(I_{n}\)。是一個非常重要的矩陣，可以用它來代表列運算、操作反矩陣等等...
  • Upper triangular matrix: A為\(n\times n\)矩陣且\(a_{ij}=0,\forall {i>j}\)，如果是\(a_{ij}=0,\forall {i\ge j}\)，對角線元素皆為0，則稱為strictly upper triangular matrix
  • Lower triangular matrix: A為\(n\times n\)矩陣且\(a_{ij}=0,\forall {i< j}\)，如果是\(a_{ij}=0,\forall {i\le j}\)，對角線元素皆為0，則稱為strictly lower triangular matrix
  • Diagonal matrix: A為\(n\times n\)矩陣且\(a_{ij}=0,\forall {i\neq j} \)，對角矩陣同時為上三角矩陣也為下三角矩陣

矩陣相等

Definition: \(A,B \in F^{m\times n}\)，A=B iff A,B當中所有元素都相等。

矩陣運算定理

assume \(A,B,C,D\in F^{m\times n}\)，\(x,y\in F\)
有些看似簡單的定理在某些題目很有用，反著想可能就會解出來，例如把O轉化成\(O\cdot A\)

  • A+B = B+A
  • (A+B)+C = A+(B+C)
  • A+O = A = O+A
  • A+(-A) = O = (-A)+A
  • (x+y)A = xA + yA
  • x(yA) = (xy)A
  • (-1)A = -A
  • \(0\cdot A\) = O
  • \(x\cdot O\) = O
  • A(BC) = (AB)C
  • \(A(B\pm C) = AB\pm AC\)
  • \((B\pm C)A = BA\pm CA\)
  • \(ABC\pm ADC = A(B\pm D)C\) ，IMPORTANT!!!
  • x(AB) = (xA)B = A(xB)
  • 若A為方陣(\(n\times n\))，則A可以和自己相乘，且可以定義\(A^0 = I_n\),\(A^k = A^{k-1}A\)
以上的性質和傳統的數字運算差不多，但矩陣有幾個性質和數字運算不同，如以下
  • AB不一定等於BA，在某些特殊條件下會等於，例如B=\(A^{-1}\)(反矩陣)
  • \(A^n = O\)不保證A = O，但A = O的話\(A^n\)必定等於O
  • \(A^2 = A\)不保證A=I或O
  • \(A\neq O,B\neq O\)不保證\(AB\neq O\)，但反過來\(AB\neq O\)保證\(A\neq O, B\neq O\)
  • \(AB = AC \)不保證B=C，不具消去性
  • \(X^n=A\)不保證X最多具有n個解，可能更多或更少
